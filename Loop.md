# Loop
what is used to reapeat a block of code until the given condition is true .

## Type of loop
--> while  
--> do while   
--> for  

-----

# While loop
while loop is used when we don't know the number of iteration in advance.

### Note 
- It is known as entry or pre-test loop.

### Syntax 
    while(condition){
        // statements
    }

### Flowchart 

[while loop](./Loop1.tldr)

-----

# Do while loop
do while loop used when we want to execute loop body once even condition is false.

### Note 
- do while loop also known as post test or exit control test.

### Syntax 
    do{
        // statement
    }while(condition);

### Flowchart 

[do while loop](./Loop2.tldr)

# for loop
for loop is used when we want to perform initialization , condition and incr/decr in single line

### Note
- for loop is the most common used loop.

### Syntax 
for(initialization; condition; incr / decr){
    // statements
}
