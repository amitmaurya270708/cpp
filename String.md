# String
String is nothing but sequence of characters. In simple, when letters are written in sequence consider as a string.

## Example 
"Lear Coding"

## Note 
- String index always starts with 0
- The last character of string is Null (\o)

## C++ provide pre-define functions to manipulate strings
- strlen()
- strrev()
- strcat()
- strcmp()
- strcpy()
- getline()       (String input function)
- push-back()
- pop-back()
