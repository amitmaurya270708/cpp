# Datatype
Datatype define the type of value means which kind of value the variable will store.    
Ex :-   
char c = 'A';     
int a = 10;   
float b = 20.5;
___
# C++ datatype
(1) Pre defined --->  integer, boolean, float, double,etc.
(2) Derived ---> function, array, pointer, etc.      
(3) User defined ---> class, structure, union, etc.
___
## (1) Pre-defined datatype :-
| Datatype | Size   | Range |  
|----------|--------|-------|
| int      | 4bytes |-2147483648 to +2147483647|
| char     | 1byte  | 0 to 255 |
| boolean  | 1byte  | true or false |
| Float    | 4byte  | -3.4 * 10^38 to 3.4 * 10^38    |
| double   | 8byte  | 

 # Type Casting 
Type casting is nothing but process of converting a variable from one datatype to another.    
EX :-    

    main(){
        int a = 10;
        float b = (float) a;
    }

# Variable 
Variable is nothing but of memory location where we store the value.

## Note :-   
1) Variable are the Case Sensitive in C++.
2) In C++, Variable must be starts with either (a-z, A-Z) or underscore(_).
3) We can't give extra space between the variable.
