## What is identifier ?
Identifier refers to the name that is used to identifie variables, function and so on.
## Example :-

    int a = 10;  // a is identifier
    
    void fun();  // fun is identifier

    class A{}  // A is identifier

-----

## What is Keyword ? 
Keyword is nothing but reserved words, whose meaning aleready defined on the compiler.
## Example :-
    int for = 10;  // it will show error because for is reserved keyword

## Note :-
- We can't use keyword as a variable & constant name .
- Keyword must be in lowercase .
