# 1) Difference batween C and C++ 
---
## C
* C supports function programming.
* C is a middle level language.
* In C &nbsp; langauge :-  
1> scanf( ) - used for input.            
2> printf( ) - used for output.
* In C we can't use inheritence. 
* File extension in C is .C
* C is a subset of C++

---
## C++
* C++ supports oops concepts.
* C++ is the high level language.
* In C++ language :-  
1> cin - used for input.  
2> cout - used for output.
* In C++ we can use inheritence.
* File extension in C++ is .cpp
* C++ is the superset of C

 # 3) Difference between Statically type & Dynamatically typed language 

## Statically 
The language which force the user to define the datatype before initialization of a variable called Statically typed language 

### Ex - 
C, C++, Java,etc


## Dynamically 
- The language in which the datatype is not compulsory and its optional called dynamically typed language

### EX - 
Python, Javascript, etc
